# Tasks-and-warehouse-system
This api is used to manage tasks and warehouses.
- Employee management system
- Task management system
- Warehouse locations
- Categories 
- Orders management system
- Products management system
## Technologies used
- Node.js
- Express.js
- MongoDB
- JWT
- Prisma
- npm
- Typescript
